# iw6x-server-launcher
![ghosts_launcher](https://user-images.githubusercontent.com/86622355/225738457-7128028e-e6b2-43d4-9666-63ffb6205fcc.PNG)

## Overview

A launcher for Call of Duty: Ghosts, designed for the iw6-mod (formerly iw6x) project. This tool simplifies the process of launching dedicated servers with custom configurations.

## Features

- **Easy Server Configuration**: Quickly create servers with custom map rotations and game modes
- **Multiple Game Modes**: Support for Multiplayer, Aliens, and Safeguard modes
- **Rank Support**: Access ranks and options that aren't available in default mode (requires non-squad mode)
- **Alterware Support**: Updated to support the newer Alterware project

## Known Issues

- **Direct Connect**: When using direct connect, you must select "Online Play" in the game menu before the connection is established
- **Auto-Update**: The automatic Alterware launcher execution on update is currently not working as it did with xlabs
